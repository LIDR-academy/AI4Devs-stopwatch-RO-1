Ultra Stopwatch - Instrucciones Rápidas
=======================================
Developer: Carlos E. Navia Torres


![Alt text](/res/ui-app-stopwatch.png)


1. Abre "index.html" en cualquier navegador moderno.
2. Usa los botones para iniciar/pausar/reanudar ("Start"/"Pause"/"Resume") y limpiar ("Clear").
   - "Clear" solo está habilitado cuando el cronómetro está en pausa o detenido.
3. Atajos de teclado:
   - [Space] → Iniciar / Pausar / Reanudar
   - [C] o [Esc] → Limpiar (solo si está en pausa)
4. El cronómetro se detiene automáticamente al llegar a 24h (24:00:00.000).
5. 100% responsivo y accesible (con ARIA y roles).
6. Estilos: solo Tailwind CSS (CDN).
7. No requiere instalación: basta con abrir el archivo.

Chatbot IA usado: ChatGPT (vía prompt de usuario)
