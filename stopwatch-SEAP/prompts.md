# ChatGPT 4.5

Rol: Eres un desarrollador web experto, especializado en HTML, CSS y JavaScript.
Tarea: Crea un cronómetro y cuenta atrás basados en JavaScript puro

Características:
- El cronómetro debe permitir iniciar, detener, continuar y reiniciar el tiempo.
- La cuenta atrás debe permitir configurar un tiempo inicial, iniciar la cuenta atrás, pausarla, continuar y reiniciarla.
- Se debe cambiar entre la funcionalidad de cronómetro y cuenta atrás con un botón arriba, con el cual clicando se cambiará entre la funcionalidad de cronómetro y la de cuenta atrás.
- La interfaz para el cronómetro debe parecerse visualmente a la imagen de referencia que te adjunto (stopwatch.png).
- La interfaz para la cuenta atrás debe parecerse visualmente a la imagen de referencia que te adjunto (countdown.png).
- Usa estilos CSS simples pero visualmente claros, similares a los de la imagen proporcionada.

Condiciones:
- No usar frameworks externos (como React o Vue), solo JavaScript puro.
- Usa únicamente HTML, CSS y JavaScript en archivos separados claramente (index.html, script.js, styles.css).
- La precisión del cronómetro debe ser al milisegundo.
- Añade comentarios al código JavaScript para explicar claramente la lógica de cada función.
- Todo el código y comentarios debe estar en inglés.

Interacción del usuario esperada:
- Un botón "Iniciar/Pausar" para controlar el cronómetro o cuenta atrás.
- Un botón "Limpiar/Reiniciar" para reiniciar el cronómetro o cuenta atrás.
- Visualización clara y grande del tiempo transcurrido o restante (formato hh:mm:ss:ms).
- Para la funcionalidad de cuenta atrás el usuario debe poner el tiempo inicial.

Diseño visual:
Basado en las imágenes que te adjunto (stopwatch.png y countdown.png). Incluye:
- Números grandes en negro sobre fondo claro (caja de visualización con bordes redondeados).
- Botones claramente diferenciados (verde para iniciar, rojo para detener/reiniciar) y con borde negro.

Entrega requerida:
Generar el código en tres archivos:
- index.html
- script.js
- styles.css
