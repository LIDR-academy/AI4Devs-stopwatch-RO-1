# PROMPT 1 - CLAUDE 3.5 SONNET

Crea una aplicación web simple que tenga la opción de cronómetro y cuenta atrás.
La aplicación debe:
1. Disponer de un selector interactivo donde indicar cronómetro o cuenta atrás.
2. Disponer de un visualizarod de tiempo en formato HH:MM:SS (que será editable si se ha cogido la opción cuenta atrás para indicar el tiempo).
3. Disponer de un botón para iniciar el cronómetro o la cuenta atrás.
4. Disponer de un botón para pausar el cronómetro o la cuenta atrás.
5. Si se pausa el temporizador, el botón de iniciar cambiará el nombre a reanudar y realizará la función de reaundar el tiempo.
6. Disponer de un botón para limpiar los datos.
7. Para la estructura, el estilo y el diseño utiliza Tailwind CSS y puedes coger como referencia la imagen stopwatch.png de la ruta res\stopwatch.png
8. Incluir todo el código HTML y CSS en un archivo llamado index.html en la carpeta stopwatch-DSR
9. Incluir todo el código Javascript en un archivo llamado script.js en la carpeta stopwatch-DSR
10. Cargar Tailwind CSS desde un CDN.

Genera el código completo y asegurate de que se pueda ejecutar haciendo doble click en el archivo index.html
