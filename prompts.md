## Objetivo: Genera un cronómetro (stopwatch) en dos archivos separados.

### Entregables

- index.html: estructura de la página con:

  - Botones Start, Stop y Reset.

  - Área visible para el tiempo en formato mm:ss:ms.

- script.js: toda la lógica del cronómetro en JavaScript puro (sin frameworks).

### Requisitos

    El HTML debe enlazar correctamente a script.js.

    Actualización del cronómetro en intervalos de milisegundos con precisión.

    CSS básico embebido en el HTML (fuente legible, botones con padding y estados hover).

    Código bien comentado y organizado.

    Solución clara, funcional y reutilizable.

### Criterios de aceptación

    Formato de tiempo exacto: mm:ss:ms (con ceros a la izquierda).

    Botones con comportamiento:

    Start: inicia/reanuda y deshabilita el botón mientras corre.

    Stop: pausa sin perder el tiempo acumulado.

    Reset: pone a 00:00:000 y deja todo listo para iniciar de nuevo.

    Sin dependencias externas.

    Formato de salida esperado

### Proporciona solo el código en dos bloques, cada uno con su título:

- index.html

- script.js
